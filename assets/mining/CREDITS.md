## What's in this folder
- `nodes/*.png` — 14 static, single-frame resource-node sprites (see contents below).
- `rock_sheet_overview_promo.png` — the pack's promo/overview image (192x96). It contains a
  showcase grid of the nodes PLUS "PREMIUM VERSION" marketing panels. It is a marketing image,
  not a clean sprite atlas.
- `LICENSE.txt` — original license text from the pack.

## IMPORTANT: no animations in the free pack
The zip advertises "Break-able Rock Animations" and "Icons" folders, but in the free version
**both contain only a README saying the content is Premium-only.** There are **no harvesting/
mining/break animation spritesheets and no icons** in this free download. Every PNG here is a
single static frame. Animated mine/deplete states require the paid Premium version.
